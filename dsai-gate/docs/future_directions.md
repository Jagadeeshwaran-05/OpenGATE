# Future Directions

Last reviewed: 2026-06-14

The next stage of DSAI-GATE should focus on making the repository easier to
trust, run, and study from.

## Priorities

1. **Reliable notebooks**
   - Add a reproducible Python environment.
   - Make maintained notebooks run from a clean kernel.
   - Remove required Colab-only paths and private data dependencies.

2. **Stronger validation**
   - Add notebook execution and schema checks to CI.
   - Improve link parsing and exclude the stale-link audit from live checks.
   - Provide one documented local validation command.

3. **Complete syllabus coverage**
   - Map every official syllabus item to a primary resource and practice set.
   - Fill known gaps in statistics, databases, calculus, and AI inference.
   - Clearly label enrichment material.

4. **Better practice**
   - Add original MCQ, MSQ, and NAT questions with explained solutions.
   - Tag official previous-year questions by topic, year, marks, and type.

5. **Clearer navigation**
   - Standardize subject pages and study paths.
   - Separate exam preparation, enrichment, and interview material.
   - Improve accessibility for diagrams and image-based notes.

## Target Outcome

A learner should be able to choose a topic, study it from a trusted source,
run a focused example, and complete relevant GATE-style practice within one
clear learning path.
