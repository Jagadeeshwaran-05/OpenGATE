# Official GATE DA Research And Resources

Last verified: 2026-06-11

This page favors official GATE pages and primary educational sources. A resource
being useful does not mean every chapter is in the DA syllabus; follow the
syllabus mapping below.

## Current Exam Facts

- The latest published syllabus is the
  [official GATE 2026 DA syllabus](https://gate2026.iitg.ac.in/doc/GATE2026_Syllabus/DA_2026_Syllabus.pdf).
- DA has 15 marks of General Aptitude and 85 marks of subject questions.
- Questions are MCQ, MSQ, and NAT. MCQs have negative marking; MSQs and NATs
  do not.
- IIT Guwahati has published the
  [2026 DA master paper](https://gate2026.iitg.ac.in/doc/download/2026/QPs/DA.pdf)
  and [answer key](https://gate2026.iitg.ac.in/doc/download/2026/Keys/DA_Keys.pdf).
- In the 2026 DA subject section, the official key identifies 23 MCQs, 14 MSQs,
  and 18 NATs.

## What The Latest Paper Emphasizes

The official 2026 paper samples all seven syllabus areas. It repeatedly requires
multi-step reasoning: translating a model into parameter counts, deriving
probabilities, tracing algorithms, checking logical statements, and evaluating
database constraints or queries.

High-value recurring families across the official 2024-2026 papers include:

- Conditional probability, expectation, common distributions, and inference.
- Matrix properties, eigenvalues/eigenvectors, projections, and PCA.
- Python behavior, asymptotic analysis, sorting, searching, trees, and graphs.
- Keys, functional dependencies, relational algebra/SQL, indexing, and OLAP.
- Regression, regularization, classification, validation, clustering, and
  neural-network fundamentals.
- Informed/uninformed search, minimax, logic, conditional independence, and
  probabilistic inference.

This is an evidence-based prioritization, not an official topic-weightage
guarantee.

## Primary Learning Resources By Subject

### Probability And Statistics

- [NPTEL: Probability for Computer Science, IIT Kanpur](https://nptel.ac.in/courses/106104233)
- [MIT OpenCourseWare: Introduction to Probability](https://ocw.mit.edu/courses/res-6-012-introduction-to-probability-spring-2018/)
- [Harvard Stat 110](https://stat110.hsites.harvard.edu/)

Prioritize conditional probability, distributions, expectation/variance,
correlation/covariance, CLT, confidence intervals, and hypothesis tests.

### Linear Algebra

- [MIT OpenCourseWare: Linear Algebra](https://ocw.mit.edu/courses/18-06-linear-algebra-spring-2010/)
- [Gilbert Strang's official linear algebra page](https://math.mit.edu/~gs/linearalgebra/)

Prioritize subspaces, matrix properties, systems, eigenvalues, projections, LU,
SVD, and PCA connections.

### Calculus And Optimization

- [MIT OpenCourseWare: Single Variable Calculus](https://ocw.mit.edu/courses/18-01sc-single-variable-calculus-fall-2010/)
- [Stanford: Convex Optimization book and course material](https://web.stanford.edu/~boyd/cvxbook/)

Use only the single-variable limits, continuity, differentiability, Taylor
series, maxima/minima, and optimization portions for core DA preparation.

### Programming, Data Structures, And Algorithms

- [NPTEL: Programming, Data Structures and Algorithms using Python](https://nptel.ac.in/courses/106106145)
- [MIT OpenCourseWare: Introduction to Algorithms](https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-fall-2011/)
- [Official Python tutorial](https://docs.python.org/3/tutorial/)

Prioritize Python semantics, stacks, queues, linked lists, trees, hash tables,
searching, sorting, divide-and-conquer, graph traversal, and shortest paths.

### Database Management And Warehousing

- [NPTEL: Database Management System](https://nptel.ac.in/courses/106105175)
- [MIT OpenCourseWare: Database Systems](https://ocw.mit.edu/courses/6-830-database-systems-fall-2010/)
- [Pearson: Fundamentals of Database Systems](https://www.pearson.com/en-us/subject-catalog/p/fundamentals-of-database-systems/P200000003546/9780137502523)

Include ER and relational models, relational algebra, tuple calculus, SQL,
constraints, normalization, file organization, indexing, transformations, and
warehouse modelling.

### Machine Learning

- [Stanford CS229](https://cs229.stanford.edu/)
- [Stanford CS229 notes](https://cs229.stanford.edu/main_notes.pdf)
- [scikit-learn user guide](https://scikit-learn.org/stable/user_guide.html)

Use implementations to support the syllabus, but retain mathematical reasoning
and from-scratch calculations for GATE-style questions.

### Artificial Intelligence

- [NPTEL: An Introduction to Artificial Intelligence, IIT Delhi](https://nptel.ac.in/courses/106102220)
- [Berkeley CS188 open textbook](https://inst.eecs.berkeley.edu/~cs188/textbook/)
- [Artificial Intelligence: A Modern Approach official site](https://aima.cs.berkeley.edu/)

Prioritize informed/uninformed/adversarial search, propositional and predicate
logic, conditional independence, variable elimination, and sampling.

## Resource Acceptance Checklist

Before adding a resource, confirm that it:

- Maps to a named syllabus item.
- Is hosted by an official institution, author, publisher, or project where
  possible.
- Is legally accessible.
- Has a stable HTTPS landing page.
- Adds value beyond an existing primary resource.
