## File Verification

The SHA-256 checksums in [`SHA256SUMS`](SHA256SUMS) were recorded when the
files were reviewed on 2026-06-11.

Verify the files locally with:

```bash
sha256sum --check PYQ/SHA256SUMS
```

When adding a new year, use the organizing institute's official master paper
and final answer key, follow the existing filenames, update this table, and
record new checksums.
